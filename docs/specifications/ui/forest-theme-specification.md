# 숲속 테마 UI 명세서

## 개요
이 문서는 MyTodoList 애플리케이션의 숲속 테마 UI 디자인에 대한 명세를 제공합니다. 자연스러운 숲속 분위기를 통해 사용자에게 편안하고 친근한 경험을 제공하는 것이 목표입니다.

## 디자인 원칙
1. **자연스러움**: 숲속의 색상과 요소를 활용하여 자연스러운 분위기 조성
2. **일관성**: 모든 UI 요소에 일관된 테마 적용
3. **가독성**: 시각적으로 명확하고 읽기 쉬운 인터페이스 제공
4. **사용성**: 직관적이고 사용하기 쉬운 UI 구성

## 색상 팔레트

### 주요 색상
- **짙은 녹색(--forest-dark-green)**: `#2c5f2d` - 헤더, 푸터, 강조 요소
- **중간 녹색(--forest-medium-green)**: `#4e9f3d` - 버튼, 경계선, 상태 표시
- **연한 녹색(--forest-light-green)**: `#97bf8f` - 보조 요소, 호버 상태
- **아주 연한 녹색(--forest-very-light-green)**: `#d0e7ca` - 배경, 비활성 요소
- **갈색(--forest-brown)**: `#654321` - 경계선, 강조 요소
- **연한 갈색(--forest-light-brown)**: `#a67c52` - 보조 요소
- **베이지(--forest-beige)**: `#f5f0e1` - 배경, 카드 호버 상태

### 기능적 색상
- **대기 상태(--pending-color)**: 연한 갈색
- **진행 중 상태(--in-progress-color)**: 중간 녹색
- **완료 상태(--completed-color)**: 짙은 녹색

## 타이포그래피
- **기본 폰트**: Roboto, sans-serif
- **헤더 크기**: 28px, 굵게(700)
- **제목 크기**: 18px, 중간 굵기(600)
- **본문 크기**: 14px, 일반(400)
- **보조 텍스트**: 12px, 일반(400)

## 컴포넌트 명세

### 1. 앱 컨테이너
- 전체 높이 차지(min-height: 100vh)
- 숲속 배경 이미지 적용
- 헤더, 메인 콘텐츠, 푸터로 구성된 수직 레이아웃

### 2. 헤더
- 짙은 녹색 배경
- 흰색 텍스트
- 숲 패턴 배경 효과(낮은 불투명도)
- 하단 경계선: 중간 녹색

### 3. 칸반 보드
- 반투명 베이지 배경
- 둥근 모서리
- 그림자 효과
- 갈색 경계선
- 가로 스크롤 지원

### 4. 칸반 컬럼
- 연한 녹색 배경
- 상단 색상 바(상태별 다른 색상)
- 제목 영역에 상태별 아이콘 표시:
  - 대기: 🌱
  - 진행 중: 🌿
  - 완료: 🌳
- 드래그 중 시각적 피드백

### 5. 칸반 카드
- 흰색 배경
- 왼쪽 경계 색상(상태별 다른 색상)
- 제목, 설명, 푸터 영역으로 구성
- 호버 시 베이지 배경으로 변경
- 드래그 중 스케일 효과
- 애니메이션 효과(페이드인)

### 6. 버튼
- **기본 버튼**: 중간 녹색 배경, 흰색 텍스트
- **보조 버튼**: 투명 배경, 녹색 경계선, 녹색 텍스트
- 호버 시 변형 효과(약간 위로 이동)
- 비활성 상태: 연한 색상, 변형 효과 없음

### 7. 푸터
- 짙은 녹색 배경
- 흰색 텍스트
- 숲 패턴 배경 효과(낮은 불투명도)
- 상단 경계선: 중간 녹색

## 애니메이션 및 전환 효과
- **페이드인**: 컴포넌트 등장 시 부드러운 페이드인 효과
- **호버 효과**: 버튼 및 카드 호버 시 변형 및 색상 변경
- **드래그 효과**: 드래그 중인 요소에 스케일 및 불투명도 변경

## 반응형 디자인
- **모바일**: 컬럼이 수직으로 쌓임
- **태블릿/데스크톱**: 컬럼이 수평으로 배치
- 모든 요소는 화면 크기에 맞게 조정

## 접근성 고려사항
- 충분한 색상 대비
- 명확한 시각적 계층 구조
- 키보드 네비게이션 지원
- 스크린 리더 호환성

## 구현 가이드라인
- CSS 변수 활용
- 모듈식 CSS 구조
- 의미 있는 클래스명
- 반응형 미디어 쿼리 사용
